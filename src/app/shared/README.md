# shared/

Piezas **reutilizables y sin estado** que se importan en muchas features.

- `components/` — componentes de UI reutilizables (botón, modal, tabla, spinner).
- `directives/` — directivas reutilizables.
- `pipes/` — pipes reutilizables (formato de moneda, fecha, etc.).
- `models/` — tipos auxiliares compartidos por la UI.

Regla: nada de lógica de negocio específica de un módulo.
